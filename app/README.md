Что я добавил в приложение помимо ТЗ:
- Сделал 3 модуля: app, data, domain.
- Перым экраном открывается не добавление пользователя, а экран со списком пользователей.
- При успешном добавлении появляется Toast
- Можно удалять пользователей из списка. Для этого нужно нажать на 3 точки в карточке. Откроется BottomSheet. И в нем нажать на удаление пользователя.
- Для открытия карточки отдельного пользователя нужно на новый экран передать его userId. Для этой передачи используется Keeper.
- Используется библиотека Coil, благодаря которой в кэш сохраняется аватарка пользователя. И она будет загружаться в последующем, даже если нет интернета.
- Добавил темную тему (но цвета там не очень красивые еще. Я подбирал цвета на ходу. А нужно полноценно отрисовать макеты в фигме, и продумать их)
- На экране со списком пользователей кнопка добавления находится поверх контента.
